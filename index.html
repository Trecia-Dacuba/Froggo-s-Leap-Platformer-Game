ABOUT THE GAME

Froggo's Leap is an original HTML5 canvas platformer built for the UPHSD College of Computer Studies WST Final Project. Play as Froggo - a determined little frog in a tiny hat - leaping across lily pads above a moonlit pond.

The game features a complete game loop: animated menu, three handcrafted levels with escalating difficulty, a level-clear screen, a You Win screen, and a Game Over screen with persistent high-score tracking.

Built entirely in vanilla JavaScript + HTML5 Canvas with procedurally generated chiptune BGM and Web Audio API sound effects - zero external libraries, zero dependencies.

GAME FEATURES

🎮

COMPLETE GAME LOOP

Animated title screen, 3 escalating levels, a level-clear screen, a You Win ending, and a Game Over screen — all with high-score tracking.

🎵

CHIPTUNE AUDIO

Unique BGM melody for each of the 3 levels generated via the Web Audio API, plus 10+ distinct sound effects for every game action.

🐸

ANIMATED CHARACTER

Froggo features squish-and-stretch physics, changing facial expressions, walking leg animation, directional flipping, and a tiny hat.

🌿

DYNAMIC PLATFORMS

Lily pads gently bob on water. Level 2 and 3 introduce moving platforms with clear visual indicators so you always know what's moving.

🐝

ENEMY AI

Hornets patrol their lily pads and scale in speed per level. Stomp them from above for bonus points, or avoid them carefully.

⭐

POWER-UPS

Shield Stars absorb one hit, Mushrooms give a speed boost — both with on-screen timer bars so you always know how long they last.

🚩

CHECKPOINTS

Mid-level checkpoint flags save your progress. Dying respawns you at the last flag you touched — not back at the start of the level.

✨

PARTICLE EFFECTS

Collectible bursts, stomp explosions, jump puffs, shield blocks, and floating score text all built with a custom particle engine.

🌄

PARALLAX BACKGROUNDS

Stars, moon, background trees, and clouds scroll at different depths creating a layered sense of 3D depth across all 3 unique level themes.

HOW TO PLAY

⌨ KEYBOARD CONTROLS

← →
Move Froggo left or right
SPACE
Jump — press again mid-air for double jump
↑ / W
Alternative jump key
A / D
Alternative move left / right
HOLD ↑
Hold jump for higher arc, tap for short hop

📱 MOBILE CONTROLS

◀ ▶
On-screen left / right buttons
▲
On-screen jump button (circular)

🏆 SCORING

ACTIONPOINTS
Collect a fly+10
Stomp a hornet 🐝+25
Reach the lotus 🌸+100 base
Level bonus+50 per level
Fly collection bonus+5 per fly

💡 TIPS

🐸 You have 3 lives — dying respawns at last checkpoint
🌿 Double jump = tap jump again while airborne
🕐 Coyote time = still jump just after walking off edge
🐝 Stomp hornets by falling on top of them
⭐ Shield absorbs one hit from hornets or falling
🚩 Touch green flags to save your position
